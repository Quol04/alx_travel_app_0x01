# alx_travel_app_0x01

## Features
- CRUD APIs for Listings and Bookings
- RESTful endpoints under `/api/`
- Interactive API documentation with Swagger

## Endpoints
- `/api/listings/`
- `/api/bookings/`
- `/swagger/` (Swagger UI)
- `/redoc/` (ReDoc UI)

## Setup
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
